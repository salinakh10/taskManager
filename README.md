# Task Manager CLI Application

## Project Description
This is a command-line interface (CLI) application that allows users to manage tasks.

## Features:
- Add a new task.
- View tasks.
- Delete a task by ID.
- Mark a task as complete.
- Save and load tasks to/from a file.
